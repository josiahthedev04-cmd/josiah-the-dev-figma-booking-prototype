# Josiah The Dev Digital Services Booking Prototype

## Project Overview

This project is a Figma booking prototype for a digital services business called **Josiah The Dev Digital Services**.

The prototype shows how a real customer can view services, choose a package, complete a booking request, review the details and receive a confirmation.

This project was created to show practical Figma UI and UX design skills through a real business use case.

## Live Prototype

Figma Prototype:
https://crane-nap-40239792.figma.site/

## Business Purpose

The purpose of this prototype is to help a digital services business collect client requests in a simple and professional way.

A customer can use the prototype to:

- View available services
- Learn what each service offers
- Choose a service package
- Fill out a booking form
- Review the booking request
- Submit the request
- See a confirmation screen
