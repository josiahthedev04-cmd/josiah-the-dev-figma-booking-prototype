# Screen List

## Desktop Screens

### 1. Home

The home screen introduces Josiah The Dev Digital Services and gives the customer a clear starting point.

Main actions:

- Book a Service
- View Services

### 2. Services

The services screen shows all available services.

Services shown:

- AI Support
- Coding
- Web Development
- Social Media Management
- Graphic Design
- Video Editing

### 3. Service Details

The service details screen explains one selected service in more detail.

Example service used:

- AI Support

### 4. Packages

The packages screen allows the customer to compare service options.

Packages shown:

- Starter Audit
- Business Launch Kit
- Content Growth Support
- Custom AI/Web Build

### 5. Portfolio Examples

The portfolio screen shows examples of work and project proof.

Examples shown:

- Canva Skills Advertising Kit
- Figma Booking Platform UI Kit
- AI Chatbot Planning Concept
- Social Media Content Pack

### 6. Booking Form

The booking form collects the customer's project details.

Fields included:

- Full Name
- Business Name
- Email Address
- Phone or WhatsApp
- Service Needed
- Package Selected
- Budget Range
- Project Timeline
- Project Details

### 7. Review Booking

The review screen allows the customer to check the booking details before submitting.

Main actions:

- Edit Details
- Submit Request

### 8. Booking Confirmation

The confirmation screen tells the customer that the request was submitted successfully.

Main actions:

- Back to Home
- View Portfolio

### 9. Contact

The contact screen gives the customer ways to contact Josiah The Dev.

Contact options:

- Email
- WhatsApp
- GitHub
- YouTube
- Social media
